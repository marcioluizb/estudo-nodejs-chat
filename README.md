# Chat - Nodejs


Chat simples utilizando:
 - express
 - socket.io
 - ejs



### Installation
Instale nodejs https://nodejs.org/en/
Instale o yarn https://yarnpkg.com/pt-BR/

baixe o repositório
Instale as dependências do projeto
```sh
$ cd estudo-nodejs-chat
$ yarn install
$ yarn start
```

Acesse no browser localhost:3000
